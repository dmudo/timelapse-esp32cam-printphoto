SOBRE O PROJETO

Estamos usando a  placa de desenvolvimento Esp32CAM e um módulo DHT11 para aferição da umidade e temperatura, a aplicação consiste em tirar uma foto em determinado período e enviar para o servidor. A ESP32-CAM, é uma placa muito pequena que une o poderoso módulo ESP32 com uma câmera OV2640 de 2MP e um slot para cartão MicroSD, entretanto nesse projeto não está sendo usado o slot de cartão,  as imagens estão sendo enviadas diretamente para o servidor, e as fotos são tiradas no intervalo de 3,33 minutos. A placa se conecta a internet via wifi e foi montada em uma case de camera IP.
